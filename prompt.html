Prompt Maestro

Prompt único de alta precisión proporcionado a los agentes de IA (Cursor y OpenCode) para generar la Landing Page de un comercio llamado "Cosecha Dorada". 

<context>
Act as an expert Front-End Development and UI/UX Design team.
Your task is to create a complete, professional, modern, and visually attractive Landing Page for a premium healthy food store called "Cosecha Dorada".
The website must transmit freshness, health, naturalness, quality, and trust.
The result must be 100% functional and ready to deploy on Vercel without any manual modifications.
</context>

<business_context>
Business Name: "Cosecha Dorada"
Concept: A premium healthy food store offering fresh seasonal fruits, natural whole foods, artisanal jams, and organic packaged products. The visual storytelling should connect the fresh fruits with the artisanal jam-making process.
Values: Natural • Healthy • Fresh • Delicious • Honest • Sustainable
Target Audience: Families, fitness enthusiasts, and health-conscious consumers seeking premium ingredients.
Tone of Voice: Friendly, professional, inspiring, and honest.
</business_context>

<branding>
Create a custom textual logo for "Cosecha Dorada" using HTML/CSS.
The logo must feel like a real premium healthy food brand and be elegantly integrated into both the header and the footer.
Do NOT use placeholder logos, basic typography, or generic text treatments like <h2>Logo</h2>.
</branding>

<language>
Prompt instructions are written in English for clarity.
However, ALL user-facing content generated for the website MUST be written in Latin American Spanish (Español Latino), including menus, headlines, products, testimonials, alt text, and SEO metadata.
The generated HTML document MUST use: <html lang="es">
The only content that must remain in English is the image-generation prompt written inside HTML comments.
</language>

<design_system>
Style: Premium Natural Market with a clean, editorial-inspired layout, generous white space, and subtle rounded elements.
Avoid: Dark themes, excessive glassmorphism, overused gradients, and generic floating cards.
Color Palette:
- Primary Orange: #FF8C00
- Secondary Orange: #F47A26
- Golden Accent: #DFAF37
- Fresh Green: #5FAF4A
- Background: #FFF8F0
- Section Background: #FFFFFF
- Primary Text: #2B2B2B
- Secondary Text: #666666
Typography (Google Fonts):
- Headings: Outfit (600, 700, 800)
- Body: Inter (400, 500)
</design_system>

<images>
Use ONLY local image paths. Do NOT use external URLs.
Do NOT generate SVG placeholders, dummy illustrations, or temporary visual blocks.
Only use the provided image paths. Do not create or assume any additional files.
Do NOT reuse the same image for multiple products. Each product card MUST display its specific image.

You MUST use the following specific paths:
- Hero Section: img/hero-fruits.jpg
- About Section: img/about-store.jpg
- Categories Section: img/products-display.jpg
- Jam Section: img/artisanal-jam.jpg
- Product Naranjas Frescas: img/product-orange.jpg
- Product Frutillas: img/product-strawberry.jpg
- Product Granola Orgánica: img/product-granola.jpg
- Product Mermelada: img/product-orange-jam.jpg

Obligatory: Immediately AFTER each <img> tag in the HTML, include an HTML comment with a highly detailed English prompt describing the perfect image for that section, so I can generate it with another AI tool. Describe professional food photography, natural golden hour lighting, and premium grocery presentation.
</images>

<required_sections>
Create the following sections:
1. Header: Logo on the left, navigation menu on the right. Sticky header with subtle shadow. Mobile hamburger menu.
2. Hero Section: Large headline, short subtitle, strong CTA button, and the fresh fruits image.
3. About Us: Short description about the store, its mission, and quality commitment. Include image.
4. Product Categories: 4 categories (Fresh Fruits, Whole Foods, Packaged Products, Artisanal Jams) using icons or images.
5. Featured Products: Grid showing premium products. It MUST strictly include these 4 items: Naranjas Frescas, Frutillas, Granola Orgánica, Mermelada Artesanal de Naranja. Each product must contain: Product image, Product name, and a Short description.
6. Why Choose Us: 3 or 4 reasons (quality, organic, sustainable, handmade).
7. Testimonials: 2 or 3 customer reviews.
8. Newsletter: Short section encouraging users to subscribe.
9. Contact: Form (front-end only) with fields for Name, Email, Message, and Submit button.
10. Footer: Logo, quick links, categories, social media icons, and copyright.
</required_sections>

<content>
- Generate all text content automatically.
- Write original, persuasive, and realistic text.
- Do NOT use emojis anywhere in the website.
- Do NOT use: Placeholder names, Generic filler content, "Lorem Ipsum", "John Doe", or "Sample Company".
- Generate realistic customer names commonly used in Spanish-speaking countries.
</content>

<interactions_and_animations>
Implement: Reveal on Scroll using IntersectionObserver, hover effect on product cards (transform: translateY(-6px)), smooth transitions (duration: 300ms), sticky header, fully functional mobile hamburger menu. Avoid heavy animations.
</interactions_and_animations>

<responsive_design>
Use a Mobile First approach. Minimum breakpoints: Mobile (320px+), Tablet (768px+), Desktop (1024px+). The layout must adapt correctly across all viewport sizes.
</responsive_design>

<accessibility_and_seo>
- All form inputs must have associated <label> tags.
- Color contrast must meet WCAG AA standard.
- Website must be fully navigable using only the keyboard.
- Use optimized <title>, meta description, proper H1-H3 hierarchy, and semantic HTML5.
</accessibility_and_seo>

<technical_constraints>
You MUST use ONLY HTML5, CSS3, and Vanilla JavaScript (ES6+).
You MUST NOT use React, Vue, Angular, Bootstrap, Tailwind, jQuery, TypeScript, or external icon libraries. Use inline SVG icons when needed.
</technical_constraints>

<output_format>
Return EXACTLY three complete code blocks in this order:
First code block: index.html
Second code block: styles.css
Third code block: main.js

The HTML file MUST reference the external CSS and JavaScript files:
<link rel="stylesheet" href="styles.css">
<script src="main.js" defer></script>

Do NOT embed CSS inside <style> tags or JS inside <script> tags within index.html.
Do not return any text before, between, or after the code blocks.
DO NOT truncate code. Provide the full, production-ready code.
</output_format>

<acceptance_criteria>
The solution is considered correct ONLY if all of the following are true:
✓ Header, Hero, About, Categories, Featured Products (with 4 specific items), Testimonials, Newsletter, Contact, and Footer exist.
✓ Responsive on mobile, tablet, and desktop using Mobile First approach.
✓ No frameworks or libraries are used (Only HTML5, CSS3, Vanilla JS).
✓ Images use local paths only.
✓ Content is realistic (No Lorem Ipsum, emojis or generic placeholders) and in Latin American Spanish.
✓ Code is complete, functional, and ready for deployment without modifications.
</acceptance_criteria>

<verification>
Before returning the final answer:
1. Verify all acceptance criteria are met.
2. Verify no forbidden technologies are used.
3. Verify that all visible website content is written in Latin American Spanish.
4. Verify that no placeholder images, SVG illustrations acting as photos, placeholder text, or unfinished elements remain.
If anything is missing or not optimal, fix it automatically. Return ONLY the final validated solution.
</verification>
